# API Gateway

Create Api endpoints.

Integration points:

- Lambda Functions
- HTTP
- Mock
- AWS Service

## Features

- API Keys
- Custom Domain names
- CLient Certificates
- Deployment Stages (dev, prod)
- Authorizers (Authentication)
- Models: Schema of requests data that the API will receive for validation purpose.
- API Documentation
- BInary Support: for sending files.
- Logs Dashboard
